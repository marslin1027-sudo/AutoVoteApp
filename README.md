# 🚀 股東e票通 - 輕鬆投自動化助手 (AutoVoteApp)

這是一套本地端自動化輔助工具。每到股東會旺季，手動一筆一筆投票、截圖總是耗時又費力。這套工具能幫你全自動完成登入、投票與截圖存檔，更具備 **E-Gift 資格掃描** 與 **Excel 批次建檔** 等強大功能，讓你輕鬆搞定電子投票，開心領取股東會紀念品！🎁

## ✨ 核心功能

* 🤖 **全自動無頭式投票**：一鍵啟動！自動掃描未投票項目，可設定預設投票意向（贊成、反對、棄權），或透過關鍵字進行精準手動投票。
* 📸 **智慧截圖存檔**：投票完成後自動進入明細頁面截圖。支援「依帳號獨立資料夾」或「合併單一資料夾（檔名加上戶名）」兩種存檔模式。另備有「單筆補圖」工具。
* 🎁 **E-Gift 資格全面掃描 (全新)**：獨立的掃描引擎，一鍵遍歷並匯出所有帳號符合「E-Gift」領取資格的清單（產出 `E-Gift領取清單.txt`）。
* 📂 **Excel 帳號批次匯入 (全新)**：內建產生 Excel 範本功能，支援大批次匯入多組帳號與身分證，免去手動建檔的麻煩。
* 🌐 **網址管理**：支援自訂網址記憶，任務完成後自動為您開啟瀏覽器。
* 🔄 **撤銷小幫手**：投錯了？沒關係！支援輸入指定股票代號撤銷，或是全自動掃描並撤銷所有已投項目。
* ⚡ **多帳號與多憑證支援**：可預先綁定多組身分證字號自動輪替執行。支援「券商網路下單憑證」、「自然人憑證」與「行動自然人憑證」。
* ⚙️ **進階速率控制**：內建 UI 可自由微調「投票速度」與「截圖速度」，適配不同電腦效能與網路環境。
* 🎉 **抽獎攔截機制**：遇到官方抽獎活動可選擇「暫停 5 分鐘手動參加」或自動略過。
* 📦 **OTA 自動更新**：內建高速多執行緒更新引擎，啟動時自動檢查 GitHub 最新版本並無縫更新，永遠保持最佳狀態。

## 🔐 安全與隱私聲明 (重要)

* 本程式為 100% 本地端執行的 Python 腳本（基於 Selenium 與 Tkinter）。
* 所有的帳號資料（身分證字號）與設定檔，皆採用**硬體綁定的高強度加密（PBKDF2HMAC + Fernet）**儲存於您個人的電腦中 `APPDATA` 路徑下。
* **絕無**任何將您的個人資料、憑證、密碼上傳至任何外部伺服器的行為。
* 所有操作皆等同於模擬真人透過瀏覽器操作，請使用者自行確保已取得代操作名單之合法授權，一切法律責任由使用者自行承擔。

## 📥 如何使用 (一般使用者)

1. 請前往專案的 [Releases](../../releases/latest) 頁面。
2. 下載最新版本的 `AutoVote_vX.X.exe` 執行檔。
3. 雙擊打開程式，首次使用請先至 **[設定]** 分頁輸入你的身分證字號，並選擇你要用的憑證登入方式（亦可使用 Excel 匯入），點擊「儲存設定」。
4. 切換回 **[自動任務]** 分頁，勾選要執行的帳號，點擊「啟動程式」即可放開雙手，讓工具替你代勞！☕

## 🛠️ 開發與建置環境 (開發者)

若您想自行審閱、修改原始碼或重新打包，請參考以下環境配置：

* **語言**：Python 3.8+
* **介面**：Tkinter
* **核心驅動**：Selenium (依賴 Microsoft Edge Webdriver)
* **打包工具**：PyInstaller

**依賴套件 (`requirements.txt`)：**
```text
selenium
cryptography
Pillow
pandas
openpyxl
requests
```

**打包指令參考：**
```bash
pyinstaller --noconsole --onefile --name=AutoVote --icon=icon.ico AutoVote.py
```
